About
왜 Tally를 만들었는가

비츠바겐(Bitsbargain)은 이커머스 셀러가 겪는 데이터 사일로 문제를 해결하기 위해 설립된 테크 스타트업입니다.

매출은 Wing에서, 광고는 광고센터에서, 정산은 엑셀에서 — 흩어진 데이터를 매일 수작업으로 취합하는 셀러의 현실에서 출발했습니다.

Tally는 이 모든 데이터를 자동 수집·통합하여 하나의 대시보드에서 실시간으로 보여주는 셀러 인텔리전스 플랫폼입니다. 직접 셀러로서 겪은 페인포인트를 기술로 해결합니다.

Automation First

수작업 제거. 데이터 수집부터 보고서 생성까지 완전 자동화

Single Source of Truth

모든 채널의 매출·비용·수익을 하나의 뷰로 통합

Battle-tested

자사 셀러 운영으로 실전 검증된 시스템

Features
셀러가 진짜 필요한 기능만
매일 반복되는 수작업을 자동화하고,
흩어진 데이터를 한눈에 볼 수 있도록 설계했습니다.

실시간 매출 대시보드

로켓·판매자 채널별 매출, 주문 수, 순이익을 실시간으로 추적. 어제 대비 변동까지 한눈에 파악합니다.

Real-time

광고 성과 분석

캠페인별 ROAS, CPC, 전환율 자동 계산. 키워드 단위 수익성 분석으로 광고비 최적화를 지원합니다.

Analytics

재고 · 사입 관리

실시간 재고 추적, 자재 소진 예상일 계산, 사입 이력 관리. 품절 리스크를 사전에 알림으로 전달합니다.

Automation

정산 자동 다운로드

정산 엑셀을 매일 자동 다운로드하고 수수료·배송비·할인 내역을 파싱하여 상품별 순수익을 산출합니다.

Finance

키워드 · 노출순위 분석

검색 트렌드, 월별 검색량, 시장 규모를 분석하고 내 상품의 노출 순위를 자동으로 추적합니다.

SEO

텔레그램 알림

매출 변동, 재고 부족, 광고 이상 등 주요 이벤트를 텔레그램으로 즉시 알림. 어디서든 실시간 대응합니다.

Notification
How It Works
복잡한 셀링 데이터, 심플하게
연동부터 인사이트까지 4단계로 완성됩니다.
1

채널 연동

쿠팡 Wing, 광고센터, 네이버 등 판매 채널을 연결합니다

2

자동 수집

매출·주문·광고·정산 데이터를 실시간으로 자동 수집합니다

3

통합 분석

채널별 데이터를 통합하여 상품 단위 수익성을 계산합니다

4

인사이트 전달

대시보드와 알림을 통해 즉시 의사결정할 수 있도록 합니다

Team
작지만 강한 팀

비츠바겐은 이커머스 실무 경험과 소프트웨어 엔지니어링을 결합한 팀입니다. 직접 셀러로 운영하며 겪은 문제를 기술로 해결합니다.

모든 기능은 실제 운영 데이터로 검증되고, 하루 수백 건의 주문을 처리하는 프로덕션 환경에서 안정성을 입증했습니다.

우리는 셀러가 데이터를 모으는 데 시간을 쓰지 않고, 데이터를 기반으로 결정하는 데 집중할 수 있도록 돕습니다.
